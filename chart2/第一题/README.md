# 习题1
     机器人使用一个可以测量 0 -3m 距离的传感器。为了简化，假定真实的距离在这个范围中均匀分布。很不幸的是，传感器会坏掉。当传感器故障时，不管传感器的锥形测最范围内实际测距结果应该是多少，其输出测距值均小于 1m 。 已知对于传感器故障的先验概率是p = 0. 01 。 
     设想机器人查询了 N次传感器，每次测量值都小于 1m。对于 N=l, 2, …,10 的传感器故障的后验概率是多少？用公式表示相关的概率模型。

## 【分析】
    传感器已经测量到测量值，且测量值小于1m。观测事件已经发生，使用贝叶斯后验估计求解后验概率。

## 【解】
    传感器测量一次测量值小于1m，求解传感器故障概率。
	设：传感器状态事件为X，传感器测量距离事件为M。	
	p(X=故障|M<1m) = p(X=故障)*p(M<1m|X=故障)/p(M<1m) = p(X=故障)*p(M<1m|X=故障)/[p(M<1m|X=正常)*p（X=正常）+p(M<1m|X=故障)*p（X=故障）]
			**第一个等号，使用贝叶斯逆概公式**	**第二个等号，使用全集分解思想，展开分母**
	已知：p(X=故障)=0.01，p(M<1m|X=故障)=1，p(M|X=正常)~U(0-3)
	代入上式得：p(X=故障|M<1m) = 0.01*1/(1/3*(1-0.01)+0.01*1) = 0.0294
